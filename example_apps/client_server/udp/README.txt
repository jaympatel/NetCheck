- The 2 programs in the directory are client and server that connect and communicate
  via UDP.
- The purpose of these programs is to locate and obtain trace of MTU issues
  occuring over the internet.
- The 2 programs use constants from cnxn_constants.h to ensure that they are
  connecting properly. The user may change these constants to alter the program.
  See the cnxn_constants.h file for more information.
- To build and run the program, call 'make' on the shell, and call 'make clean'
  to remove the binary files once done using the programs.
